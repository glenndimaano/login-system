# Social Login

> Using Passportjs Multiple Strategy

## Build Setup

```bash
# install dependencies
$ npm install

# server localhost:3000
$ npm run dev
```

### .env

```
PORT=3000
MONGODB_URI=mongodb://localhost:27017/TEST
MONGODB_CLOUD_URI=w=majority
FACEBOOK_APP_ID=
FACEBOOK_APP_SECRET=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
```
